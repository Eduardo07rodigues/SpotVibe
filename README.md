# 🎵 Spot Vibe

O **Spot Vibe** é um player de música desenvolvido com **HTML, CSS e JavaScript**, onde o usuário escolhe sua emoção ("vibe") e recebe uma playlist com músicas que combinam com aquele momento.

O objetivo do projeto foi praticar manipulação do DOM, eventos em JavaScript e a criação de um player de áudio funcional com uma interface moderna.

---

## ✨ Funcionalidades

- 😊 Escolha da emoção (vibe):
  - Feliz
  - Triste
  - Raiva
  - Ok
  - Apaixonado

- 🎶 Playlist personalizada para cada emoção.
- ⏯️ Reproduzir e pausar músicas.
- ⏭️ Avançar para a próxima música.
- ⏮️ Voltar para a música anterior.
- 🔊 Controle de volume.
- ⏱️ Barra de progresso da música.
- 📱 Interface simples e responsiva.

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📚 O que aprendi

Durante o desenvolvimento deste projeto pude praticar:

- Manipulação do DOM;
- Eventos em JavaScript;
- Controle do elemento `<audio>`;
- Atualização dinâmica da interface;
- Manipulação de arrays e objetos;
- Organização de arquivos em um projeto Front-end;
- Responsividade utilizando CSS.
